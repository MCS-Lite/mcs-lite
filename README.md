## Install

```
$ npm i mcs-lite-ui mcs-lite-icon mcs-lite-theme --save
```

## Development

```
$ npm install # Install all external dependencies of each package.
```

```
$ npm run clean # Remove all node_modules
```

## Building all mono-packages

```
# design => icon => theme => ui
$ ./tasks/build.sh
```

## Testing

```
$ ./tasks/test.sh
```

## mcs-lite-ui

### Development

```
$ npm start
```

### Testing

```
$ npm run test:watch
```

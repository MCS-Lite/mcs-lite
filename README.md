## Getting Started

```
$ npm run clean # Remove all node_modules
$ npm install # Install all external dependencies of each package.
```

## Testing

```
$ ./tasks/test.sh
```

## mcs-lite-ui

### Development

```
$ npm start
```

### Testing

```
$ npm run test --watch --coverage
```

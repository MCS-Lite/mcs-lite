# API 参考文件

MCS Lite 支援透过 **WebSocket** 与 **HTTP RESTful API** 两种方法上传与读取资料，其支援的协定与内文格式定义如下表

| WebSocket | HTTP RESTful API |
| :---| :--- |
| 上传资料：JSON | 上传资料：CSV |
| 读取资料：JSON, CSV | 读取资料：JSON, CSV |

更多详细介绍，请参考下节说明。

* ### [上传资料](/mcs_lite_api/update_datapoint.md)
* ### [读取资料](/mcs_lite_api/read_datapoint.md)
